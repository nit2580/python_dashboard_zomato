# 📊 Zomato Data Dashboard (Python Project)

An interactive data visualization dashboard built using **Python**, showcasing insights from Zomato restaurant data. This project combines **hvPlot, Panel, Matplotlib, and Seaborn** to create a clean and interactive UI with multiple visualizations.

---

## 🚀 Features

- 📌 Interactive dashboard using Panel
- 📊 Multiple visualizations:
  - Bar Charts
  - Line Charts
  - Heatmaps
  - Ridge Plots
  - Hexbin Plots
  - Violin / Distribution Plots
- 🔍 Top 5 analysis for better insights
- 🎛️ Sidebar with insights and filters
- 🧹 Cleaned dataset for accurate results
- 📈 KPI indicators for quick overview

---

## 🛠️ Tech Stack

- Python 🐍
- Pandas
- hvPlot
- Panel
- Matplotlib
- Seaborn
